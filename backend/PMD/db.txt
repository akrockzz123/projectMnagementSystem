-------------------------------------------------------------
-- API
    -- Users 
        only Admin can access 
        -- type : GET 
            - /users -> get all users
            - /{UserID}/getAll -> get All projects of a user
            - /{UserID}/getAllProjects -> get all projects done by a user (both active and inactive)
            - /{UserID}/getactiveprojects -> get all active projects of a user
            - /{UserID}/getinactiveprojects -> get all inactive projects of a user
        -- POST 
            - /add -> Add a user 
            - /{UserID}/delete -> delete a user 
            - /{UserID}/assign/ -> assign a project to the user
            - /{UserID}/update/ ->  Update the User 
        all users can access 
            - /about -> details of users
    -- Project
        only Admin can access 
            -- GET 
                - /project/getallprojects/ -> get all projects 
                - /project/{ProjectID} -> get a specific project
                - /project/getall -> get all projects (both active and inactive)
                - /project/getallactive -> get all active projects 
                - /project/getallinactive -> get all inactive projects
            -- POST 
                - /project/addproject -> Add a new project 
                - /project/remove -> remove a project
